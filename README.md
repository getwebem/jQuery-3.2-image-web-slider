# jQuery Slider
### **Getting started**
### Technologies Used
#### HTML5, CSS3, jQuery 3.2
 - Run  :  `git clone https://github.com/getwebem/picSlider.git`
 - Run  :  `cd picSlider`
 - Run :  `Brackets .`
 - Open :  `Index.html`
 - Browser:  `Live Preview`
